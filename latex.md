## Text-mode symbols

### Quotes
```latex
`single quotes'
``double quotes''
```

` this is a GRAVE ACCENT (U+0060)

' this is an APOSTROPHE {single quote; APL quote} (U+0027)

### Comments
```latex
% percent sign starts a line comment
```
